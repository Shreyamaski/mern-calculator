# MERN Calculator Web App

A responsive calculator app built with React, Express, and MongoDB. It includes history storage, theme switching, button sounds, and smooth animations.

## Project Structure

```text
Calculator/
  client/
  server/
  package.json
```

## Quick Start

1. Install root tools:

```bash
npm install
```

2. Install app dependencies:

```bash
npm run install:server
npm run install:client
```

3. Create `server/.env` from `server/.env.example`.

4. Start both apps:

```bash
npm run dev
```
